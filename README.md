📌 Description

This project is a simple secure login system made using Python and SQLite. Users can register and log in securely. Passwords are stored in encrypted form using bcrypt.

✨ Features

* User Registration

* User Login

* Password Hashing

* SQLite Database

* Secure Authentication

🛠 Technologies Used

* Python

* SQLite

* bcrypt

* Google Colab

▶ How to Run

* Open Google Colab.

* Install bcrypt using !pip install bcrypt.

* Run all the code cells.

* Choose Register or Login.
